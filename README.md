## Instructions
1. git clone https://github.com/maestroh/streamer
2. cd streamer
3. cd frontend
4. npm run build
5. cp -r build ../backend
6. cd ../backend
7. echo "/path/to/your/music" >> streamer.config
8. npm install -g
9. streamer
10. http://localhost:3001
